# 지금 상황

요즘 현타와서 밍기적 거리는 중<br/>
첫직장인데 3년 채우기도 전에 망하기 직전이라서 빨리 이직준비해야한다는 압박감 <br/>
쁘라스로 <br/>
할 의욕이 완전히 사라진 상황임 <br/>

이직할려고하는데 지금 포트폴리오가 이 사이트 하나뿐이라 뭘 할지 감도 안오고<br/>
딱히 하고싶은건 없으니 밍기적 거리긴 하는데<br/>

지금 이 사이트도 완성된건 아니고 그냥 돌아가긴 해서 내비두고 업데이트만 하는데<br/>
새로운 기능 넣을정도로 의욕이 안생긴다<br/>

그래서 지금 사둔 책이나 읽으면서 있다가 <br/>
지금 이정도 포트폴리오로 JS로 이직해보고 안된다 싶으면 <br/>
퇴직금이나 회사 망해서 퇴사된거로 처리되면 실업급여나 뭐 어떻게든해서 <br/>
시간좀 벌면서 한두달 준비했다가 이직이나 해야지 뭐...<br/>

게다가 지금 회사 망하기 직전이라 직원이 나 혼자. <br/>
그말은 즉슨 지금까지 하던 업무가 줄어서 망한거긴한데 그 작업을 혼자한다!<br/>
와! <br/>
급여는 그대론데 업무는 배 ! <br/>
그래서 더 열받아있는 상황이긴함. <br/>
